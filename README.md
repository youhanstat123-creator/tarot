#  Tarot Project (타로 프로젝트)

##  프로젝트 소개

사용자가 타로 카드를 선택하여 결과를 확인할 수 있는 웹 서비스입니다.
회원가입 및 로그인 기능을 포함한 풀스택 프로젝트입니다.

---

##  사용 기술

* **Backend**: Java (Spring Boot)
* **Frontend**: HTML, CSS
* **Build Tool**: Gradle
* **Deployment**: Render

---

##  주요 기능

* 회원가입 / 로그인
* 타로 카드 선택 기능
* 타로 결과 출력
* 간단한 UI 인터페이스

---

##  배포 주소

👉 [https://tarot-c6af.onrender.com]

---

##  프로젝트 구조

* `src` : 백엔드 로직
* `gradle` : 빌드 설정
* `Dockerfile` : 배포 설정

---

##  느낀 점

* 로그인 기능 구현을 통해 인증 흐름을 이해할 수 있었습니다.
* 프로젝트 배포(Render)를 경험하며 실제 서비스 흐름을 익혔습니다.
